Recursion

What is Recursion? Recursion is a process(or most cases a function) that calls itself. This works by invoking the same function with a different input until you reach your base case. The base case is a condition when the recursion ends. With that in mind, there are two parts of recursion that are important:

  1. The base case - where the recursion ends
  2. The different input - this input is changed and passed to the function during the recursive call

Consider the simple example below that counts down from an integer passed to zero:

  1. const countDown = (num) => {
  2.     // checks wether the recursion has reached the base case
  3.     if (num <= 0) {
  4.         return 'Done';
  5.     }
  6.  
  7.     // placed just to show case the different inputs
  8.     console.log(`'Countdown input': ${num}`);
  9.  
  10.     // decrement the input by 1
  11.     num--;
  12.  
  13.     // pass the modified input to the recursive call
  14.     return countDown(num);
  15. }

Let's check some examples below:

Sum Range Problem
Problem: Write a function called 'sumRange', which accepts an integer num. The function should return the sum of numbers in the range between 1 to the integer passed.

To visualize it, the function will look like: sumRange(3) // 6 

  1. const sumRange = (num) => {
  2.     // base case
  3.     if (num === 1) {
  4.         return 1;
  5.     }
  6.  
  7.     // each recursive call will return a value up the call stack to the first return
  8.     return num + sumRange(num - 1);
  9. }
Factorial Problem
Problem: Write a function called 'factorialNum', that returns the multiplication of all numbers between 1 and num.

To visualize it, the function will look like: factorialNum(4) // 24 

  1. const factorialNum = (num) => {
  2.     if (num <= 1) {
  3.         return 1;
  4.     }
  5.  
  6.     return num * factorialNum(num - 1);
  7. }
Bonus Concept: Helper Method Recursion

This concept is basically used whenever you need to return data populated using a helper recursive function. In the below example, we demonstrate how you can populate an array of odd numbers from an array input using a helper recursive function.

Problem: Write a function called 'collectOddValues', that returns odd numbers from an array.

To visualize it, the function will look like: collectOddValues([1, 2, 3, 4, 5, 6, 7, 8, 9]) // [1, 3, 5, 7, 9] 

  1. const collectOddValues = (arr) => {
  2.     // initialize an array to house the results(odd numbers)
  3.     let result = [];
  4.  
  5.     const helper = (helperInput) => {
  6.         // base case condition
  7.         if (helperInput.length === 0) {
  8.             return result;
  9.         }
  10.  
  11.         // check if number is odd, modulus 2 will give remainder
  12.         if(helperInput[0] % 2 !== 0) {
  13.             result.push(helperInput[0]);
  14.         }
  15.  
  16.         // remove the first item in the array
  17.         helperInput.splice(0, 1);
  18.  
  19.         // pass new array to recursive call
  20.         return helper(helperInput);
  21.     }
  22.  
  23.     helper(arr);
  24.  
  25.     return result;
  26. }
Bonus Concept: Pure Recursion

This concept is does not rely on a helper function but rather in merging a set of data over and over as the function is executed in the call stack.

For this concepts there are a few things to consider:

  1. For arrays, use methods like slice, the spread operator, and concat that make copies of arrays so you do not mutate them
  2. Remember that strings are immutable so you will need to use methods like slice, substr, or substring to make copies of strings
  3. To make copies of objects use Object.assign, or the spread operator
Problem: Write a function called 'collectOddValues', that returns odd numbers from an array.

To visualize it, the function will look like: collectOddValues([1, 2, 3, 4, 5, 6, 7, 8, 9]) // [1, 3, 5, 7, 9] 

  1. const collectOddValues = (arr) => {
  2.     // initialize an array to house the results(odd numbers)
  3.     let result = [];
  4.  
  5.     // base case condition
  6.     if (arr.length === 0) {
  7.         return result;
  8.     }
  9.  
  10.     // check if first item in the array is odd
  11.     if (arr[0] % 2 !== 0) {
  12.         result.push(arr[0]);
  13.     }
  14.  
  15.     /**
  16.      * Here we merge the result array to the return value of the recursive call.
  17.      * 
  18.      * This is the same as: result = result.concat(collectOddValues(arr.slice(1)));
  19.     */
  20.     result = [...result, ...collectOddValues(arr.slice(1))];
  21.  
  22.     return result;
  23. }
